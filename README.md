# Braintree-PPS-Cloud

Documentation for Braintree PPS

## GitHub Pages

This repository is configured with GitHub Pages to host documentation. The site is automatically deployed from the `docs` folder whenever changes are pushed to the `main` branch.

Visit the documentation site at: https://braintree-by-vox.github.io/Braintree-PPS-Cloud/

### Local Development

To view the site locally, simply open `docs/index.html` in your web browser.
